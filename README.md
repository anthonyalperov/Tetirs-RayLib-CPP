# Tetris (C++ / Raylib)

A fully custom Tetris clone built in C++ using Raylib.
Features:
- 7-bag randomizer
- Hold system
- Ghost piece
- Hard drop
- Line clear animations
- High score saving
- Full SFX
- Main menu + controls overlay

Controls:
Left/Right - Move
Up - Rotate
Down - Soft drop
Space - Hard drop
C - Hold
H - Controls
P - Pause
