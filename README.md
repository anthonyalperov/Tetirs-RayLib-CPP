# Tetirs-RayLib-CPP

**Why I built this**
Honestly, I just got tired of trying to play Tetris and being bombarded by ads every two minutes. Instead of dealing with trackers and pop-ups, I figured it would be more fun to just build my own version where I can actually focus on the game.

**What this is**
This is a straightforward, no-nonsense Tetris clone. I specifically chose to skip using a full-blown game engine for this because I wanted to keep things "under the hood."

The goal was to move away from the "hacky" coding habits of my earlier projects and focus on:

**Clean Code**: Writing structured, readable, and "intended" professional code from the start.

**Memory Management**: Getting hands-on practice with references and efficient memory usage in C++.

**Data Handling**: Leveling up my string manipulation within a real-time environment.

It’s built using **RayLib** to keep it fast, lightweight, and—most importantly—completely ad-free.
